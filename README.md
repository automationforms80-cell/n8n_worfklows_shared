# AI Advertising Workflows

Open-source AI workflows for generating ads, extracting customer insights, and automating creative production with tools like n8n, Claude, and Nano Banana.

These workflows are mainly built around advertising use cases: turning websites, brand assets, customer language, and product inputs into structured ad concepts, copy, and visuals.

If you want the simpler hosted version without setting up workflows, API keys, and prompts manually, that’s what I’m building at [blumpo.com](https://blumpo.com).

---

## What this repository includes

This repository contains DIY workflows for:

- generating AI ad creatives from a website, logo, and image inputs
- extracting customer insights from websites, Reddit, and other online sources
- creating ad copy with Claude
- generating visuals with Nano Banana
- automating parts of the ad creation process in n8n
- experimenting with AI-powered creative workflows

Each workflow has its own folder with a separate README and setup instructions.

---

## Who this is for

This repo is for marketers, founders, agencies, and automation builders who want to:

- run AI advertising workflows themselves
- customize prompts and logic manually
- use their own API keys
- learn how AI ad generation pipelines can be structured
- adapt the workflows for their own creative or marketing process
