# AI Ad Workflow with Claude + Nano Banana

This is the DIY workflow I built for generating ads from a website, logo, and image inputs.

If you want the simpler hosted version without setting everything up manually, that’s what I built at [blumpo.com](https://blumpo.com).

## What it does

- scans website inputs
- pulls brand context like visuals, tone, and value props
- creates ad copy with Claude
- generates visuals with Nano Banana
- gives you ready ad outputs faster than doing it manually

## Who this is for

This repo is for people who want to run the workflow themselves in n8n, tweak prompts manually, and use their own API keys.

## DIY workflow vs Blumpo

### This repo
- self-hosted in n8n
- manual setup
- your own API keys
- customizable

### Blumpo
- no workflow setup
- easier generation flow
- more automation around brand / context
- built for faster use
