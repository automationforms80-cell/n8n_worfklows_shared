# AI Ecommerce Ad Generator with Claude + Nano Banana

This is a DIY n8n ecommerce ad workflow inspired by what we’re building at [blumpo.com](https://blumpo.com).

It generates ecommerce-style ad creatives from a website, product image, and brand inputs. The workflow helps turn product pages, product visuals, and website copy into structured ad concepts, product-focused messaging, and generated ad creatives.

## What it does

- takes a website and product image as input
- analyzes the product and brand context
- extracts useful product messaging from the website
- creates ecommerce ad copy with Claude
- generates product ad visuals with Nano Banana
- outputs ad-ready creative concepts and visuals

## Who this is for

This workflow is for ecommerce brands, store owners, marketers, agencies, and automation builders who want a DIY ecommerce ad generation setup in n8n using their own API keys and custom prompts.

For the simpler hosted version, visit [blumpo.com](https://blumpo.com).
