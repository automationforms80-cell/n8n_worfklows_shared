# AI Ad Generator with Claude + Nano Banana

This is a DIY n8n version of an AI ad generation workflow inspired by what we’re building at [blumpo.com](https://blumpo.com).

It generates ad creatives from a website, logo, and image input. The workflow scans the website, extracts useful brand context, analyzes the uploaded visual asset, creates ad copy with Claude, and generates the final ad creative with Nano Banana.

## What it does

- takes a website, logo, and image input
- extracts brand messaging and visual context
- creates ad copy and creative direction with Claude
- generates ad visuals with Nano Banana
- outputs a ready ad concept and generated creative

## Who this is for

This workflow is for founders, marketers, agencies, and automation builders who want to run their own AI ad generation workflow inside n8n.

For the simpler hosted version, visit [blumpo.com](https://blumpo.com).
