# AI Ad Generator with Claude + Nano Banana

This n8n workflow generates ad creatives from a website, logo, and image input.

It scans the website, extracts brand context, creates ad copy with Claude, and generates visuals with Nano Banana.

Use it if you want a DIY setup where you can edit prompts, use your own API keys, and customize the workflow manually.

For the simpler hosted version, visit [blumpo.com](https://blumpo.com).
